THE CONSTRAINT THAT CHANGED

Code was expensive.
That shaped everything. The constraint is gone.

AI coding tools now deliver 38–42% cost reductions. The constraint that shaped SaaS — expensive code — no longer exists. Time to separate what was valuable from what was just an artifact.

What stays
ICP discipline, problem identification, go-to-market, feedback loops — the business model disciplines are the real value.
What dies
The parameterized monolith — one codebase, same UI for everyone, customization via config. An artifact of the constraint.
What's coming
Custom-generated software — specific code per customer. Same ICP discipline, new tech layer. This is what we're building.
